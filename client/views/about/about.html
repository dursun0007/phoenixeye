What it is

Phoenix Eye is an image forensics tool, it serves as a GUI for phoenix, which you can download and use for free!

By examining the outputs from phoenix, one can determine if an image has been digitally altered (photoshopped) or not. The output images will test for certain aspects which we would expect to see on natural (unaltered) images. You can find more information about the analyses in the Information page.

What you can do with the Phoenix Eye:

  • Learn about image forensics algorithms.
  • See if the cat picture your co-worker posted could be photoshopped.
  • Make sure your sext-buddy isn't overly attached to the liquify tool.

What it is not

Phoenix Eye will not place a verdict on the image, but just present the outputs for your own analysis. All of these analyses have false-positives and false-negatives, and should only be used as a piece of information complementing a greater base of knowledge about the particular image.

Also please understand that I'm just one guy doing this as a hobby. I've done my best to document and provide references for the algorithms, but this isn't NSA shit we're talking about here.

What you cannot do with the Phoenix Eye:

  • Become an image forensics guru.
  • Use it to prove Obama was from the planet Nubunon all along.
  • Uncover a global Illuminati conspiracy by analyzing images found at a forum post back in 2005.

Story

What was to become the Phoenix Eye today was started back in 2010 when I saw the Body By Victoria analysis by Neal Krawetz of Hackerfactor. I wanted to implement some of the algorithms in PHP and make a website for the FTW! 2010 competition, so I did. It was a hard lesson to learn that PHP was far from the best choice to implement image processing algorithms. The code took a pathetically long time to process an image of any reasonable size in PHP (upwards of 1.5 minutes for 3-4 Megapixel image).

So I switched to C++, and after starting over many times I decided to name it phoenix. I decided to make a GUI as a website/service, so here we are. Feel free to check out the source on GitHub.

Technologies Used

phoenix command tool is programmed in C++ and uses OpenCV and Boost libraries.

Phoenix Eye uses Node.js for backend, MongoDB as storage, AngularJS for frontend.
