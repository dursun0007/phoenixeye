Digital Image Forensics

Phoenix Eye is an image forensics tool, it serves as a GUI for phoenix, an open source image forensics tool which you can download and use for free!

Phoenix Eye runs common image forensics algorithms on the submitted image. It also extracts important information from images.

Features:

  • Error Level Analysis
  • Copy-Move (Clone/Stamp) Forgery Detection
  • Average Distance
  • Luminance Gradient
  • HSV and Lab Histograms
  • 3D RGB Colorspace
  • Extracts EXIF data
  • Extracts Quantization Tables
  • Estimates JPEG Quality
